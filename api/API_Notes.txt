Firstly, we will create our first user.
We can login or register user which can be done in 2 ways:
1. Inside our user API [Not Recommended] [Not best practice because login process is a different word and 
   to create more secure applications we should create a separate route like "auth.js"]
2. Creating separate API (auth.js) [Recommended]

Auth.js: Route for login and register.

We cannot directly save our password inside the DB. Hence, we will encrypt the password and then save.
There are several ways to encrypt the password that we can google.
For this application we will CryptoJS. (Documentation: https://cryptojs.gitbook.io/docs/) (npm website: https://www.npmjs.com/package/crypto-js)
